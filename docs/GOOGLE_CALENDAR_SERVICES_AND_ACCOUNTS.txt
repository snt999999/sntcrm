СОЛНЦАНЕТ — Google Calendar: услуги и аккаунты

1. Услуги и суммы в Google Calendar

В этой сборке Cloudflare Functions передают в Google Calendar расширенное поле «Услуги и суммы».
В событии календаря должно отображаться:

Услуги и суммы:
1. Название услуги — материал: ... — сумма ₽
2. Название услуги — материал: ... — сумма ₽
Итого: ... ₽

Для авто данные берутся из строк «Авто услуги».
Для архитектуры берутся «Услуга», «Плёнка/Пленка», «м²», «Общая стоимость».

2. Важно по Apps Script

Чтобы Google Calendar точно показывал новый красивый блок, открой Google Apps Script календаря и замени код на файл:

google-calendar-sync-appsscript-v32.gs

После замены кода нажми:
Deploy → Manage deployments → Edit → New version → Deploy

URL Web App можно оставить тот же. В Cloudflare должны остаться:
GOOGLE_CALENDAR_SYNC_URL или GOOGLE_CALENDAR_IMPORT_URL
GOOGLE_CALENDAR_SYNC_TOKEN или GOOGLE_CALENDAR_IMPORT_TOKEN
